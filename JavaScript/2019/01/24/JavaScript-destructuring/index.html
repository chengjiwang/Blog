ES6 - 解構賦值

解構賦值(Destructuring Assignment)

解構賦值，用於提取(extract)陣列或物件中的資料，並像鏡子一樣將資料賦值到另一方的陣列或物件。

陣列解構

  • 將陣列的資料賦值到變數

  • 解構和其餘參數一起使用

物件解構

  • 取出物件內的值到單一變數上

  • 取出其中一個值，並附在新的變數名稱上

  • 解構和其餘參數一起使用

  • 解構用在函式的參數
