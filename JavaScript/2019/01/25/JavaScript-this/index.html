JavaScript - this

this 會隨著 function 調用(執行)的方式不同而不一樣。只存在於function 裡面，全域不會有。

function 調用的四種方法如下：

1. 作為函式(function)

在嚴謹模式下為 undefined 。
非嚴謹模式下為全域根物件(在瀏覽器為 window 物件，在 node.js 為 global 物件)。

2. 作為物件的方法(methods)

會回傳執行時的所屬物件

3. 作為建構式(constructor)

this 為建立的新物件

4.透過 call()或 apply()方法間接調用
