React 學習筆記

React是一個 JavaScript 函式庫，用來處理view 的部分。
它定義畫面如何變化，而不是去操作DOM。

JSX

  • JSX 讓我們使用類似HTML的語法去建立 React 樹節點。
  • JSX 是一個輔助的語法，幫助我們開發，執行前會被轉譯成 JavaScript。

HTML Tags vs. React Components

  • React 可以 render HTML tags 和 React components。
  • 在JSX 裡使用lower case命名(HTML tags)或 upper case命名(React components) 去做區分。

JavaScript Expressions

1.Attribute Expressions

使用JavaScript expression 設定屬性的值，用{ }取代 ‘ ‘

2.Boolean Attributes

若省略屬性的值，JSX 會將屬性的值以 true 對待。

props

  • 使用 props 可以將外面的資料傳入元件，props透過 JSX 語法裡的屬性來傳遞。
  • 元件透過 this.props 取得資料。
  • 不要在元件內部修改 props。
